## 介紹
此程式協助你下載 iLearning 教材影片，並透過 OpenAI-WhisperAPI 轉錄成逐字稿。
## 使用方法

1. [下載壓縮檔](https://github.com/MO7YW4NG/CYCU-iLearning-Video-Transcription/releases)
2. 解壓縮後生成 `CYCU-iLearning-Video-Transcription` 資料夾
3. 開啟 `app.exe` 後登入即啟用!
